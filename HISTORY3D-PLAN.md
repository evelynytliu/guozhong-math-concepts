## 課本結構（康軒 2026，第二單元 臺灣的歷史(上)）

- 序章：學習歷史的 3 個 W（What 歷史事件/紀錄/研究、Why、How 史前vs歷史時代、紀年法：西元/干支/年號/民國）
- 第 1 課 史前文化與原住民族
  - 1-1 從考古認識史前文化：文化層；舊石器＝長濱文化（八仙洞，5萬-5千年前，打製石器、已知用火、採集狩獵漁撈）；
    新石器＝大坌坑(早)、圓山(貝塚)、卑南(石板棺、人獸形玉器)（7千-2千年前，農業、磨製石器、製陶、畜牧）；
    金屬器＝十三行（2千-4百年前，煉鐵、人面陶罐、玻璃手環/瑪瑙珠/銅錢＝對外貿易）。文化命名用發現地小地名。
  - 1-2 原住民族：南島語族（分布馬達加斯加～復活節島）；傳說（泰雅巨石起源、布農洪水+大螃蟹；賽夏矮靈祭矮人傳說）；
    名稱演變：清（生番/熟番，依納稅服勞役）→ 日治（平埔族/高砂族）→ 民國（山地同胞 → 民國80年代改稱原住民族，現認定16族）。
- 第 2 課 大航海時代各方勢力在臺灣
  - 時間軸：1604 荷入澎湖被沈有容勸退 → 1624 荷占南臺灣築熱蘭遮城 → 1626 西占雞籠 → 1642 荷驅西 →
    1652 郭懷一事件 → 1662 鄭成功逐荷 → 1683 施琅攻臺。
  - 2-1 東亞海上貿易：大航海時代（香料絲綢）；明海禁、倭寇/海商；顏思齊開墾雲林嘉義、鄭芝龍獨霸海上；
    日本稱臺灣「高山國」；葡占澳門、西占馬尼拉；「福爾摩沙」；VOC 1602 成立、總部巴達維亞、1624 轉臺灣。
  - 2-2 各方競爭：熱蘭遮城(Zeelandia，大員/安平)＋普羅民遮(Provintia，赤崁)；西班牙雞籠淡水聖多明哥城（紅毛城）；
    日本鎖國 → 西人縮編 → 1642 荷驅西；荷蘭經濟＝國際貿易轉運站（出口稻米糖鹿皮）、召募漢人開墾、
    稅重 → 1652 郭懷一事件；鄭成功 1661 鹿耳門登陸圍熱蘭遮城、1662 荷降；承天府、軍屯（寓兵於農，新營/左營地名）；
    鄭經＋陳永華（文教、全臺首學孔廟）；清海禁 → 鄭氏走私貿易（出口糖鹿皮）；1683 施琅攻臺。
- 第 3 課 大航海時代臺灣原住民與外來者（3-1 與歐洲人互動、3-2 與鄭氏互動）
- 第 4 課 清帝國時期的行政治理（4-1 初期治臺政策、4-2 行政區劃演變、4-3 對外事件與施政改革）
- 第 5 課 清帝國時期的農商發展（5-1 農業、5-2 與中國貿易、5-3 開港後）
- 第 6 課 清帝國時期的社會與文化（6-1 移墾社會、6-2 西方文化再傳入、6-3 原住民社會文化變遷）

## 產品設計

### 核心玩法（讓名詞記得住）
- 每課 → 1~3 個「3D 場景」，每個場景是一段**分幕推進的故事**（stage by stage）：
  鏡頭飛到定點 → 生動旁白（對 12 歲男生說話的語氣）→ 場景裡有**可點擊的發光熱點**，
  點了翻出「名詞卡」（名詞＋一句人話解釋＋記憶鉤子）。
- **名詞卡＝收集品**：全部收集到才能進下一幕（圖鑑感、遊戲感）。
- 場景結尾「快問快答」3~5 題，答錯回鍋（沿用全站精神），全對 → 拿到該場景徽章。
- 進度、收集到的名詞、徽章存 localStorage（v1 先本機；之後可比照 quiz-storage 上 Supabase）。

### 技術
- three + @react-three/fiber + @react-three/drei（需 npm install，**要網路**）。
- 低多邊形（low-poly）程序化建模 + 暖色打光 + 霧氣，走「可愛遊戲風」，不用外部 3D 模型檔。
- 場景元件 client-only（dynamic import, ssr:false），避免 SSR/WebGL 問題；iPad Safari OK。
- 路由：`/history`（hub，星圖/島嶼式關卡選擇＋徽章牆）、`/history/[sceneId]`（場景播放器）。
- 資料：`src/content/history/`（types.ts、scene-*.ts、index.ts）——名詞卡、旁白、quiz 純資料；
  3D 視覺：`src/components/history/`（player、每場景一個 R3F 元件）。
- 進度：`src/lib/history-storage.ts`。
- 入口：首頁加一張「歷史 3D 場景館」卡（社會色系）；`/subject/social` 內容清單也掛進去
  （在 `subject-content.ts` 加 kind）。

### 場景規劃（前兩課＝本次實作，5 個場景）

**第 1 課**
1. `s1-cave`「時光洞穴：往下挖就是往回走」——照課本文化層概念，鏡頭沿著考古坑往下降：
   幕1 八仙洞考古現場（名詞：考古遺址、文化層、史前時代）→ 幕2 舊石器・長濱（洞穴營火：打製石器、已知用火）→
   幕3 新石器（聚落：大坌坑、圓山貝塚、卑南石板棺+玉器、磨製石器、製陶、農業）→
   幕4 金屬器・十三行（煉鐵爐、人面陶罐、貿易品）→ 快問快答。
2. `s1-austronesian`「南島大遷徙」——海面群島+獨木舟航行（南島語族分布）→ 傳說之森（泰雅巨石、布農洪水大螃蟹）→
   名稱時光廊（生番熟番→平埔/高砂→原住民族16族）→ 快問快答。

**第 2 課**
3. `s2-age-of-sail`「大航海時代：全世界都來了」——大海圖場景，帆船逐站航行：
   歐洲人東來（澳門/馬尼拉/福爾摩沙）→ 海禁與倭寇海商 → 顏思齊鄭芝龍 → VOC 與澎湖沈有容 → 1624 落腳大員 → 快問快答。
4. `s2-two-forts`「雙城記：熱蘭遮 vs 聖多明哥」——臺灣島 3D 地圖南北飛行：
   熱蘭遮城+普羅民遮 → 西班牙雞籠淡水（1626）→ 鎖國+1642 荷驅西 → 荷蘭轉運站經濟與郭懷一（1652）→ 快問快答。
5. `s2-koxinga`「國姓爺來了」——鹿耳門海戰圍城（1661-1662）→ 承天府與軍屯（新營左營）→
   鄭經陳永華孔廟 → 走私貿易與 1683 施琅 → 快問快答。

**之後各課（hub 顯示「即將開放」佔位卡）**
- 第 3 課：`s3-sinckan`「教堂與獵場」（新港文書、傳教、鹿皮貿易）＋`s3-conflict`「衝突與共存」（原住民 vs 鄭氏軍屯）。
- 第 4 課：`s4-crossing`「渡臺悲歌」（渡臺禁令、械鬥背景）＋`s4-reform`「從消極到建省」（牡丹社、沈葆楨、劉銘傳）。
- 第 5 課：`s5-water`「水圳與良田」＋`s5-ports`「一府二鹿三艋舺」＋`s5-tea`「茶糖樟腦出國門」（開港後）。
- 第 6 課：`s6-frontier`「移墾江湖」（羅漢腳、械鬥、民變）＋`s6-mackay`「馬偕與新事物」＋`s6-pingpu`「平埔族的改變」。
